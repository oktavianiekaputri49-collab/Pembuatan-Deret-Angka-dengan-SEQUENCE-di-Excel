📊 Pembuatan Deret Angka dengan SEQUENCE di Excel
Deskripsi
Proyek ini berfokus pada pembuatan deret angka secara otomatis menggunakan fungsi SEQUENCE di Microsoft Excel.

Dataset
Data terdiri dari:
- Deret angka berurutan (contoh: 1 sampai 100)

Tools & Fungsi yang Digunakan
- Microsoft Excel
- Fungsi SEQUENCE untuk menghasilkan angka otomatis

Hasil
- Deret angka berhasil dibuat secara otomatis
- Tidak perlu input manual
- Data lebih cepat dan efisien dihasilkan

Tujuan Proyek
- Memahami penggunaan fungsi SEQUENCE
- Melatih otomatisasi data di Excel
- Membuat data berurutan dengan lebih efisien

Insight
Fungsi SEQUENCE sangat membantu dalam membuat data berurutan secara otomatis tanpa perlu mengetik satu per satu.

Penutup
Proyek ini merupakan latihan dasar dalam otomatisasi pembuatan data menggunakan Excel.
